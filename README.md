# Snowden-Visual-Novel
A narrative visual novel following Edward Snowden's journey -  from NSA contractor to whistleblower. Built with Ren'Py.

# 👁️ Classified: The Snowden Files

> *"The greatest fear I have regarding the outcome of these 
> disclosures is that nothing will change."* — Edward Snowden

**Classified: The Snowden Files** is a story-driven visual novel 
that reconstructs the true events surrounding one of the most 
significant whistleblowing cases in modern history.

## 🕵️ About the Story

Step into the shoes of Edward Snowden — a former NSA contractor 
and CIA employee who, in 2013, made the decision that would change 
the world forever. Navigate through morally complex choices as you 
relive key moments:

- 🏢 Working inside the NSA's surveillance apparatus
- 💻 Discovering the PRISM mass surveillance program
- ✈️ The escape to Hong Kong and contact with journalists
- 📰 The leak to The Guardian and Washington Post
- 🌍 Asylum in Russia — and life as a fugitive

Every choice you make reflects the real ethical dilemmas Snowden 
faced: **loyalty vs. conscience, security vs. freedom, silence vs. truth.**

## 🎭 Key Features

- Branching narrative with multiple endings based on your decisions
- Historically accurate events reconstructed from public records, 
  Snowden's memoir *"Permanent Record"*, and journalistic sources
- Original soundtrack evoking tension and surveillance-era paranoia
- Distinct visual style inspired by classified documents aesthetic
- Multiple playable perspectives (Snowden, journalists, NSA officials)

## ⚠️ Disclaimer

This visual novel is a **fictional dramatization** based on publicly 
available information. It is created for educational and artistic 
purposes. All dialogue is fictional and does not represent the actual 
words.

## 🛠️ Built With

- Python 3.x
